# qmware2-install
```
cd qmware2-install
```

## Run it

```
sudo bash start.sh
```

## Stop it

```
sudo bash stop.sh
```

Stop and remove all images
```
sudo bash stop.sh --rmi all
```

## Update QMWare 2

```
bash update.sh
```
