# qmware2-install

# Setup
Clone repository
```
git clone https://github.com/jsranko/qmware2-install.git
```

```
cd qmware2-install
bash setup.sh
```

## Run

```
sudo bash start.sh
```

## Stop

```
sudo bash stop.sh
```

Stop and remove all images
```
sudo bash stop.sh --rmi all
```

## Update QMWare 2

```
bash update.sh
```
